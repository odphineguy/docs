## Direct Messages

Send one-on-one messages to individual drivers:

1. Select a driver from the roster
2. Type your message
3. Driver receives notification in their app
4. View message history and read receipts

## Broadcast Messages

Send announcements to multiple drivers at once:

- **All Drivers** - Company-wide announcements
- **Active Drivers** - Only drivers currently on duty
- **By Region** - Drivers in specific areas
- **By Vehicle Type** - Sprinter drivers, semi drivers, etc.

## System Alerts

Automatic notifications for:

| Alert Type | Description |
|------------|-------------|
| Critical | Immediate attention required |
| Warning | Potential issues to monitor |
| Info | General information updates |

## Sound Notifications

Enable audio alerts for critical messages so dispatchers never miss important updates.
