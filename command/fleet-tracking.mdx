## Live Fleet Map

The fleet map displays all vehicles in real-time with:

- **Vehicle markers** showing current position
- **Heading indicators** showing direction of travel
- **Status colors** indicating driver state
- **Route lines** for active deliveries

## Driver Status

| Status | Description |
|--------|-------------|
| Active | Driver is on an active task |
| Idle | Driver is available but not moving |
| En Route | Driver is traveling to pickup/delivery |
| Offline | Driver app is not connected |

## Geofencing

Set up geofenced zones for:
- **Hubs** - Warehouses and dispatch centers
- **Customer locations** - Frequent pickup/delivery sites
- **Restricted areas** - Zones requiring alerts

Receive automatic notifications when vehicles enter or exit defined zones.
